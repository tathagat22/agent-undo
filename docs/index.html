undo

Undo anything your AI agent does

When you let an AI agent loose on your machine, undo records every change it makes to the real world — and reverses all of it with one command.

GitHub npm crates.io 
$ walkback watch                 # arm it — any agent's changes are now reversible
$ walkback diff                  # see exactly what it did
$ walkback rollback              ✓ rewound to cp001

Works with any AI agent

Anythingwalkback watchCursor, Copilot, Windsurf, Aider, custom scripts — snapshots, then watches the filesystem.
Any CLI agentwalkback run -- <cmd>Wraps the command; snapshots first, reversible after.
Any MCP clientnpx @walkback/coreThe agent drives undo through 16 MCP tools.
Claude Codewalkback protectNative hook — auto-checkpoints every session, zero effort.

What it reverses

📁 Files

Create / modify / delete / dirs / symlinks / permissions, byte-perfect & crash-safe. Plus redo and selective per-file revert.

🔍 walkback diff

A PR-style view of exactly what the agent changed — review before you keep it or reverse it.

🌐 Network calls

Runs the compensating request (the DELETE that undoes a POST, the refund that undoes a charge). Dry-run gated.

✉️ Email

Honest hold-and-release: the email is a draft that has gone nowhere. Cancel = a true unsend.

☁️ Cloud & databases

Records the command that reverses it (terraform destroy, inverse SQL). Works with any tool, not a hardcoded few.

🛡️ Trustworthy

Crash-safe, lock-protected, property-tested for byte-for-byte round-trips. Green on Linux, macOS & Windows.
